About me:

Cheers! I'm Maiia, a UI Designer, who's enjoying the process of researching and creating. I am very passionate about design and I feel, that I can make very nice designs that are totally user friendly and responsive. I have studied the whole Front-end part for a long time (have worked in this field) and a bit of a Backend so I know quite well what stages the web application should pass to appear on the screens of users.

In my work I use Adobe Suite and Sketch App, and prototyping to show how the website or application should work. I think that using a so common Bootstrap grid (12 columns approach) is the best strategy in a workflow.

I know how to transform mockups from Photoshop to a valid HTML and CSS code with a bit of a JQuery or JavaScript

I consider myself as a responsible, friendly and passionate person. I do enjoy learning new and working with creative people and challenging tasks.
